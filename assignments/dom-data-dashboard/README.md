# 📘 Assignment: DOM Data Dashboard

## 🎯 Objective

Build a small browser dashboard using JavaScript that displays and filters data. You will practice selecting DOM elements, rendering lists, and responding to user input events.

## 📝 Tasks

### 🛠️	Render Student Data

#### Description
Use the provided `data.json` and starter HTML to render all student records into the page.

#### Requirements
Completed program should:

- Load the provided data into JavaScript
- Render each student as a card inside the results container
- Show each student's name, grade, and score


### 🛠️	Add Live Filtering

#### Description
Add interactivity so users can filter the displayed students by typing in the search box.

#### Requirements
Completed program should:

- Listen for input events from the search field
- Filter records by name in a case-insensitive way
- Re-render the results so only matching students are shown
